# ui-chrome-extension
Chrome MIDI controller extension for Soundcraft UI

currently supporting Mackie Control Protocol

### V-POT ASSIGN overlay:
EQ, GATE, COMP, FX-SEND, AUX-SEND
### GLOBAL VIEW overlay:
GAIN, AUXSENDS, FXSENDS, PLAYER, SETTINGS
